Generación del PDF de la orden de trabajo

Así se verá el documento que entrega el sistema al cliente

Cada orden de trabajo puede imprimirse o enviarse como PDF profesional: con logo del negocio, datos del cliente, detalle del servicio, repuestos y total. El sistema genera este documento de forma automática para que el técnico o la recepción lo entreguen al cliente sin tener que escribir nada a mano. A continuación, una muestra de cómo se ve ese PDF.
